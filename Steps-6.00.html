Create databases in SQL Server 6
The following steps will only need to be performed in SQL Server Management Studio (SSMS)
  • Right-click on Databases, then select New Databases.
  • Name the database (i.e., AIVideo_Staging)
  • Click Ok to create the database. Refresh database to be able to locate it.


Use this step for creating all three layers we are looking to create.

AIVideo_Staging

AIVideo__ODS

AIVideo__DataWarehouse

Once you've created these databases, you can move on to the next step!
